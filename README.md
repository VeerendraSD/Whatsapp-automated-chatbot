# Whatsapp-automated-chatbot
A Node.js WhatsApp automation bot for an aquarium business featuring Groq AI integration for pet care advice and Google Sheets for order management

## Karnataka Aquarium WhatsApp Bot (MVP)
An automated WhatsApp assistant built with whatsapp-web.js to streamline customer inquiries and orders.

🚀 Features
AI Pet Assistant: Integrated with Groq (Llama 3.1) to provide real-time pet care advice via the pet [question] command.

Automated Order Flow: A state-managed conversation tree that collects item selection, quantity, and delivery address.

Google Sheets Integration: Automatically logs confirmed orders to a spreadsheet for fulfillment.

Product Discovery: Interactive menus for Pets, Accessories, and Recommendations.

🛠️ Tech Stack
Runtime: Node.js

Library: whatsapp-web.js

LLM API: Groq SDK (Llama 3.1)

Database: Google Sheets API v4

Authentication: LocalAuth for persistent WhatsApp sessions.


This project is currently a functional MVP. I am working on expanding the depth of the product categories and integrating a real-time inventory database.
